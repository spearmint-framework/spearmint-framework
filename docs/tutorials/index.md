# Tutorials

Learn Spearmint by building real experiments from scratch.

## Available Tutorials

### [Getting Started](getting-started.md)
Create your first experiment and understand the basic workflow. **Start here** if you're new to Spearmint.

**You will learn:**
- How to install Spearmint
- How to create a simple experiment
- How to run experiments with different configurations
- Basic configuration management

**Time:** 15 minutes

---

### [Configuration Basics](configuration-basics.md)
Master configuration management with YAML files, dynamic values, and typed models.

**You will learn:**
- Loading configurations from YAML files
- Using dynamic values for parameter sweeps
- Creating type-safe configurations with Pydantic
- Nested configuration binding

**Time:** 20 minutes

---

### [Building a FastAPI Experiment](fastapi-experiment.md)
Build a production-ready FastAPI service with shadow testing and observability.

**You will learn:**
- Integrating Spearmint with FastAPI
- Shadow testing in production
- Accessing experiment results
- MLflow integration for tracking

**Time:** 30 minutes

---

## Learning Path

1. **Complete beginner:** Start with [Getting Started](getting-started.md)
2. **Understand configurations:** Continue to [Configuration Basics](configuration-basics.md)
3. **Build real applications:** Move to [FastAPI Experiment](fastapi-experiment.md)

After completing these tutorials, explore the [How-To Guides](../how-to/index.md) for specific tasks and the [Technical Reference](../reference/index.md) for detailed API documentation.
